    OUR GAMES

    SURVIVAL REALM

    Survival Realm is a seperate server active everyday. It features a rank-less environment, meaning there's no one with greater rank than others. Equality is the moto here. It also features custom in-game proximity voice chat with music and custom seasons to enhance player's experience.

    ROLEPLAY

    Roleplay realm is a seperate server with fully configured lobby and many worlds to choose from. It runs on Minecraft 1.16.5 version for optimal performance and is planned to get updates soon! Spoiler: It will update to the latest MC version!
